- Initial Flutter web application structure
- Firebase Firestore integration for media storage

- Firebase Hosting configuration
- Responsive home screen with artist selection
- Individual portfolio screens for each artist
- Media category filtering (Stories, Photos, Writing)
- Media grid layout with responsive design
- Media detail dialog with full information
- Comprehensive documentation:
  - README.md with project overview
  - FIREBASE_SETUP.md with Firebase configuration guide
  - SAMPLE_DATA.md with test data instructions
  - DEPLOYMENT.md with deployment guide
  - CONTRIBUTING.md with development guidelines

- Firestore indexes configuration

- Web manifest for Progressive Web App support

- View portfolios for Joanna Ooms and Eleanor Ooms
- Browse media items by category

- Beautiful gradient UI with purple/pink theme

- Timestamp display for each item
- Support for images, audio, and documents
